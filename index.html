Zermelo-Fraenkel Set Theory

Axioms

  1. The axiom of extensionality states that two sets are equal if they have the same elements.

    $$ \forall x\, \forall y [\forall z(z \in x \iff z \in y) \implies x = y] $$

  2. The axiom of regularity or foundation states that every non-empty set \(x\) has a member \(y\) whereby \(x\) and \(y\) are disjoint.

    $$ \forall x (x \ne \varnothing \implies \exists y \in x (y \cap x = \varnothing)) $$

    For instance, the set \(\{1, 2, 3, 4\}\) is disjoint from its member \(1\), since (in set theory, at least), '1' is simply shorthand for \(\{\varnothing\}\).1

  3. The axiom of separation is necessary to avoid Russell's Paradox, which is an inconsistency in Frege's formulation of set theory. For Frege, there is always a set that can be defined by its inclusion of only the terms which satisfy a particular formula. Formally written, Frege's definition is:

    $$ \exists x\, \forall y [F(y) \implies (y \in x)] $$

    The problem with Frege's definition is that it merely assumes the existence of the superset \(x\); the axiom of separation in Zermelo-Fraenkel, on the other hand is conditional on its existence:

    $$ \forall x\, \exists y\, \forall z [((z \in x) \land F(z)) \implies (z \in y)] $$

  4. The power set axiom defines a 'power set' as a set containing all the subsets (including \(\varnothing\)) of another set.

    $$ \forall x\, \exists y\, \forall z (z \subseteq x \implies z \in y) $$

    For example: the power set of \(\{a,b,c\,\}\) is:

    $$ \{\{a\},\{b\},\{c\},\{a,b\,\},\{a,c\,\},\{b,c\,\},\{a,b,c\,\},\{\varnothing\}\} $$

  1. In ZFC, integers are taken as shorthand for members of the sequence \(\varnothing,\{\varnothing\},\{\{\varnothing\}\},\{\{\{\varnothing\}\}\},...\)
